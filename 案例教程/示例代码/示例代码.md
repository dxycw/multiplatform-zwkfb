# 示例代码

* 本节描述了如何使用本框架的示例代码。

## 一、Multiplatform

### 1、文本(String,......)函数

“文本()”函数用于创建文本组件，和“Text()”函数用法相同，请参考位置“[安卓x.组合.材质3.文本](../../zwkfb/src/commonMain/kotlin/%E5%AE%89%E5%8D%93x/%E7%BB%84%E5%90%88/%E6%9D%90%E8%B4%A83/%E6%96%87%E6%9C%AC.kt)”。

* 简单的文本示例
```kotlin
    文本("Hello World!")
```

### 2、按钮(单击回调,......){...}函数

“按钮()”函数用于创建按钮组件，和“Button()”函数用法相同，请参考位置“[安卓x.组合.材质3.按钮](../../zwkfb/src/commonMain/kotlin/%E5%AE%89%E5%8D%93x/%E7%BB%84%E5%90%88/%E6%9D%90%E8%B4%A83/%E6%8C%89%E9%92%AE.kt)”。

* 简单的按钮示例
```kotlin
    按钮(
        单击回调 = {
            // 点击回调代码
        }
    ){
        文本("按钮")
    }
```
